# Guide to Japan
This project is a guide to Japan. It includes 4 main pages: home, food, culture and attractions
## Technologies used
HTML
CSS
JavaScript 
## How to Use
1. Open index.html in a browser
2. Click in the navbar to navigate through the different pages
## Features
1. YouTube video + interactive google map
2. Timeline 
3. 2 webpages showing top attraction sites for toursists and best foods in Japan
